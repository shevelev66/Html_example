Примеры

Задача 6: Известен плановый P и фактический F, показатели выпуска      продукции предприятия.Определить процент выполнения плана.

ММ :

Дано : P-плановый (вещ, пер.) F- фактический (вещ, пер.)

Найти : PR- процент выполнения плана (вещ, пер.)

Связь : PR:=F*100 / P (вещ. пер.)

ДУ : P <> 0
